# 2026-06-17

## 修复：骑手注册 500 错误 (Column 'status' cannot be null)

- **问题**：`POST /api/auth/register` 骑手注册时报 `DataIntegrityViolationException: Column 'status' cannot be null`
- **根因**：`UserService.register()` 创建 `RiderDetail` 时使用 `new RiderDetail()` 无参构造，未设置 `status` 字段。`rider_details.status` 列定义为 `NOT NULL`，导致插入 NULL 失败。
- **修复**：
  1. 改用 `new RiderDetail(user.getId())` 参数化构造（该构造已设置 `status = "offline"`）
  2. 在 `RiderDetail` 实体中给 `status` 字段添加 `= "offline"` 默认值，防御性编程
