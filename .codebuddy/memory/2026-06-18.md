# 2026-06-18

## RiderService orElseThrow 报错修复
- **问题**：Spring Boot 3.5.15 中 `@Lock` + `@Query` 与 `Optional` 返回类型冲突，导致 `orElseThrow()` 无法解析。
- **修复**：
  - `OrderRepository.findByIdWithPessimisticLock()` 返回类型从 `Optional<Order>` 改为 `Order`
  - `RiderService` 中用 `if (order == null)` 替代 `.orElseThrow()`
